继承与原型链

创建对象的方式

  • 1. 使用语法结构创建对象
    •         
            var obj = {name: 'leinov'}
  • 2. 使用构造器创建对象
    •         
            function Person (name, age) {
                this.name = name;
                this.age = age;
            }
            Person.prototype.getName = function() {
                return this.name;
            }
            var p = new Person('leinov', 18);
            console.log(p.getName());
  • 3. 使用Object.create创建对象

    • Object.create()方法创建一个新对象，使用现有的对象来提供新创建的对象的__proto__

            
            var obj = {name:'leinov'} // obj --> Object.prototype ---> null
            var obj2 = Object.create(obj); // obj2 -> obj -> Object.prototype ---> null
            console.log(obj2.__proto__ === obj) // true

    可以使用Object.create(null)创建一个空对象 不包含任何属性方法 不继承Object.prototype

  • 4. 使用class关键字创建对象
    •         
            class Animal {
                constructor(type) {
                    this.type = type
                }
            }
            var a1  =  new Animal('cat');
